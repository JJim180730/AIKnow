# 十表数据查看器使用说明

## 📖 简介

基于ag-Grid的交互式表格查看器，支持在线查看、筛选、排序、搜索《史记》十表的结构化数据。

**在线访问**: `docs/special/table_viewer.html`

---

## ✨ 主要功能

### 1. 表格选择
- 下拉菜单选择10个表格中的任意一个
- 默认加载：018_高祖功臣侯者年表（143位列侯数据）

### 2. 快速搜索
- 在顶部搜索框输入关键词
- 实时搜索所有列的内容
- 支持部分匹配

### 3. 列排序
- **单列排序**: 点击列标题
- **多列排序**: 按住Shift键点击多个列标题
- 升序 → 降序 → 取消排序

### 4. 高级筛选
- 点击列标题右侧的菜单图标（≡）
- 支持筛选条件：
  - 包含（Contains）
  - 不包含（Not Contains）
  - 等于（Equals）
  - 不等于（Not Equal）
  - 开始于（Starts With）
  - 结束于（Ends With）
  - 空白/非空白
- 支持"且"、"或"逻辑组合

### 5. 列管理
- **调整列宽**: 拖拽列标题边缘
- **重置列**: 点击"重置列"按钮恢复默认状态
- 所有列自动适应容器宽度

### 6. 数据导出
- **CSV格式**: 点击"导出CSV"按钮
- **Excel格式**: 需要ag-Grid企业版（当前使用CSV替代）
- 导出文件包含当前筛选和排序结果
- 文件命名格式: `表格名_时间戳.csv`

### 7. 分页
- 默认每页显示50行
- 可选：20/50/100/200行
- 底部分页导航

### 8. 响应式设计
- 自动适配桌面、平板、手机屏幕
- 移动端优化的触摸操作

## 🎯 使用场景示例

### 场景1：查找特定功臣
1. 选择"018 高祖功臣侯者年表"
2. 在快速搜索框输入"曹参"
3. 查看曹参的封地（平阳）、户数（16600户）、功勋

### 场景2：筛选高封户功臣
1. 选择"018 高祖功臣侯者年表"
2. 点击"侯功"列的筛选菜单
3. 设置条件"包含：万"
4. 查看封户数超过万户的功臣

### 场景3：对比诸侯国
1. 选择"014 十二诸侯年表"
2. 使用搜索框快速定位特定国家
3. 按年代列排序，查看时间顺序

### 场景4：导出研究数据
1. 应用筛选条件（如：只显示前200年之后的数据）
2. 点击"导出CSV"
3. 在Excel或Python中进一步分析

## 🛠️ 技术细节

### 技术栈
- **ag-Grid Community v31.0.0**: 企业级表格组件
- **原生JavaScript**: 无需框架依赖
- **响应式CSS**: 适配多种设备

### 支持的JSON格式

#### 格式1：多section结构（三代世表）
```json
{
  "table_info": {...},
  "sections": [
    {
      "section_title": "...",
      "columns": ["列1", "列2", ...],
      "rows": [
        {"列1": "值1", "列2": "值2", ...}
      ]
    }
  ]
}
```

#### 格式2：单表结构（其他9表）
```json
{
  "table_info": {
    "id": "018",
    "title": "高祖功臣侯者年表",
    "columns": ["国名", "侯功", ...]
  },
  "rows": [
    {"国名": "平阳", "侯功": "...", ...}
  ]
}
```

### 性能优化
- **虚拟滚动**: 大数据量表格流畅显示
- **按需渲染**: 只渲染可见行
- **分页加载**: 减少初始渲染时间

## 🔧 故障排查

### 问题1：数据加载失败
**原因**: JSON文件路径错误或文件不存在
**解决**:
1. 检查 `docs/special/data/tables/` 目录下是否有JSON文件
2. 运行 `python scripts/publish_tables_data.py` 重新发布数据

### 问题2：表格显示空白
**原因**: ag-Grid CSS未加载或浏览器兼容性问题
**解决**:
1. 检查网络连接（ag-Grid从CDN加载）
2. 使用现代浏览器（Chrome/Firefox/Safari最新版）

### 问题3：中文显示乱码
**原因**: 文件编码问题
**解决**: 确保JSON文件为UTF-8编码
