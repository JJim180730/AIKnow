## 一、章节特点

**027_天官书**是《史记》八书之一，全面记录了古代天文星象学说及其占验方法：

### 内容特点
1. **星宿密度极高**：全章涉及上百个星宿、星官名称
2. **天文术语专业**：五星（岁星/荧惑/填星/太白/辰星）运行规律
3. **干支古名齐全**：十二地支古名（摄提格/单阏/执徐等）全部出现
4. **时长类型多样**：天体周期、历史跨度、观测持续时间等
5. **思想制度交织**：天文制度、阴阳五行思想并存
6. **生物象征丰富**：苍龙/白虎/朱雀/玄武四神，封豕/雄鸡等异兽

### 标注难点
- 星宿名称与地名同形（如"长沙"既是星宿也是地名）
- 干支古名易误作人名（如"昭阳作鄂"→天干地支纪年法）
- 时长与纪年混淆（"十岁"="十年时长" vs "十岁"="十岁那年"）
- 数量边界模糊（"三四百里"是一个整体还是两个数量？）

## 三、核心修正详解

### 【关键发现1】时长消歧遗漏（A8规律）

**问题描述**：天官书涉及大量天文周期和历史跨度，均属时长类型，但标注时遗漏了`|时长`消歧标注。

**修正案例**：

| 位置 | 原标注 | 正确标注 | 语境 |
|------|--------|---------|------|
| L271 | `三〖%十岁〗一小变` | `三〖%十岁\|时长〗一小变` | 天运周期 |
| L271 | `〖%百年〗中变` | `〖%百年\|时长〗中变` | 天运周期 |
| L271 | `〖%五百载〗大变` | `〖%五百载\|时长〗大变` | 天运周期 |
| L271 | `上下各〖%千岁〗` | `上下各〖%千岁\|时长〗` | 时间跨度 |
| L276 | `〖%二百四十二年〗之间` | `〖%二百四十二年\|时长〗之间` | 历史跨度 |
| L284 | `〖%十五年〗彗星四见` | `〖%十五年\|时长〗彗星四见` | 观测持续时间 |
| L284 | `久者八〖%十日〗` | `久者八〖%十日\|时长〗` | 持续时间 |
| L284 | `〖%三十年〗之间` | `〖%三十年\|时长〗之间` | 战乱持续时间 |
| L288 | `〖$数十年〗` | `〖%数十年\|时长〗` | 持续时间（误标数量→时间） |
| L294 | `〖%百年〗之中` | `〖%百年\|时长〗之中` | 历史跨度 |

**规律总结**：
- "X岁/年一Y"结构 → 时长（如"三十岁一小变"）
- "X岁/年之间"结构 → 时长（如"百年之间"）
- "久者X"结构 → 持续时间（如"久者八十日"）
- "数X年" → 数量+时长双重标注（"〖%数十年|时长〗"）

**特殊案例**：L288的"数十年"原本错误标注为数量类`〖$数十年〗`，实际应为时间类`〖%数十年|时长〗`。"数"在此为不定数量词，但整体是时间概念。

---

### 【关键发现2】原文完整性错误（标注铁律违反）

**严重问题**：标注文件擅自将原文"诸吕"改为"吕氏"，违反最高优先级规则。

**详细分析**：

- **原文（docs/original_text/027_天官书.txt）**:
  ```
  汉之兴，五星聚于东井。平城之围，月晕参、毕七重。诸吕作乱，日蚀，昼晦。
  ```

- **标注文件（修复前）**:
  ```
  〖'汉〗之兴，〖!五星〗聚于〖!东井〗。〖=平城〗之围，〖!月晕〗〖!参〗、〖!毕〗七重。〖&吕氏〗〖[作乱〗
  ```

- **修复后**:
  ```
  〖'汉〗之兴，〖!五星〗聚于〖!东井〗。〖=平城〗之围，〖!月晕〗〖!参〗、〖!毕〗七重。〖&诸吕〗〖[作乱〗
  ```

**违反规则**：
- ❌ 删除原文字符"诸"
- ❌ 插入原文没有的字符"氏"
- ✅ 正确标注应为`〖&诸吕〗`（"诸吕"在《史记》中是指吕后家族势力的专有称呼）

**语义解释**：
- "诸吕" = 吕后的诸兄弟子侄，汉初外戚势力总称
- "吕氏" = 吕姓氏族，范围更广
- 虽然语义接近，但**标注铁律禁止任何字符改动**，必须使用原文字符

| 客星 | 10+ | 彗星、天狗、蚩尤之旗、景星... |

**全部正确标注为`〖!天文〗`**

### 2. 分野系统

星宿与地理分野对应关系：

```
〖!角〗、〖!亢〗、〖!氐〗，〖=兗州〗。
〖!房〗、〖!心〗，〖=豫州〗。
〖!尾〗、〖!箕〗，〖=幽州〗。
```

**规律**：星宿名〖!〗+ 地名〖=〗的对应关系全部正确标注

### 3. 天文思想制度辨析

| 类型 | 标注 | 示例 |
|-----|------|-----|
| 天文制度 | `〖^〗` | 七政、五行、阴阳、日月、二十八舍 |
| 天文思想 | `〖_〗` | 天道、天下（政治概念）、君子 |
| 天文现象 | `〖!〗` | 日蚀、月蚀、日晕、月晕、云气 |

**区分标准**：
- 制度：可观测、可计算的天文系统（七政、五行）
- 思想：抽象哲学概念（天道、天人之际）
- 现象：具体天文事件（日食、月食）

### 规律A8（时长消歧）深度解析

**天官书中的时长类型**：

1. **天体周期**：
   ```
   三〖%十岁|时长〗一小变  ← 天运小周期
   〖%百年|时长〗中变      ← 天运中周期
   〖%五百载|时长〗大变    ← 天运大周期
   ```

2. **历史跨度**：
   ```
   〖%二百四十二年|时长〗之间  ← 春秋242年
   〖%三十年|时长〗之间        ← 秦末战乱持续30年
   上下各〖%千岁|时长〗        ← 历法推算跨度
   ```

3. **观测持续时间**：
   ```
   〖%十五年|时长〗彗星四见  ← 秦始皇时15年间4次彗星
   久者八〖%十日|时长〗      ← 彗星出现最长持续80日
   ```

4. **政治事件时长**：
   ```
   〖[诛〗〖~夷狄〗者〖%数十年|时长〗  ← 汉武帝征伐夷狄持续数十年
   ```

**判断要诀**：
- 能否替换为"持续了X年" → 是：时长
- 能否替换为"第X年" → 是：纪年
- 天官书中99%的"X年"都是持续时间，不是纪年点
