### B. 原文字符增删修复（5处）- 严重错误

这是本轮反思最重要的发现，违反了标注铁律。

| 行号 | 错误类型 | 原文 | 错误标注 | 正确标注 | 说明 |

| L127 | 删除字符 | 日冬至 | 〖%冬至〗 | 〖%日冬至〗 | 错误删除了"日"字 |

**错误根源分析**：
- 标注者误认为律书中的"阴阳"都应标注为"阴气阳气"

- 这类错误违反了"只能添加标注符号，不得修改原文字符"的铁律

### 新发现的问题模式

**问题模式A63：擅自增减原文字符的标注错误**

**错误类型**：
1. **擅自增字**：将"阴"/"阳"标注为"阴气"/"阳气"（增加"气"字）
2. **删除字符**：将"日冬至"标注为"冬至"（删除"日"字）
3. **拆分增字**：将"阴阳气"拆成"阴气""阳气"（增加一个"气"字）

**检测方法**：
```bash
python scripts/lint_text_integrity.py NNN
```
查看"插入（标注多字）"和"删除（标注少字）"类型差异

**修复原则**：
- 标注只能添加 `〖TYPE〗` 符号，不得增删改原文字符
- 去除标注后的纯文本必须与原文逐字相同
- 如需说明，使用消歧标注：`〖^阴阳气|阴阳之气〗`

**预防建议**：
1. 标注前仔细对照原文字符
2. 标注后使用lint_text_integrity.py验证
3. "实质差异"≠消歧标注，需区分真实错误和合理消歧
