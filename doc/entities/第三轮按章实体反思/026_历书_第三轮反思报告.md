## 一、干支古名标注遗漏（56处）

### 问题描述
根据**规律A57**（干支纪年古名标注规则），历书中使用的干支古名（岁阳、岁阴）必须标注为天文类〖!〗。

**天干古名（岁阳，10个）**：
- 甲→焉逢、乙→端蒙、丙→游兆、丁→彊梧、戊→徒维
- 己→祝犁、庚→商横、辛→昭阳、壬→横艾、癸→尚章

**地支古名（岁阴，12个）**：
- 子→困敦、丑→赤奋若、寅→摄提格、卯→单阏、辰→执徐、巳→大荒落
- 午→敦牂、未→协洽、申→涒滩、酉→作鄂/作噩、戌→阉茂/淹茂、亥→大渊献/汁洽

### 遗漏模式
历法表格中干支古名组合格式：`〖!天干古名〗〖!地支古名〗〖%年号〗`

**错误示例**：
```
〖!横艾〗涒滩〖%始元元年〗  → 缺少地支古名标注
```

**正确标注**：
```
〖!横艾〗〖!涒滩〗〖%始元元年〗
```

## 二、原文完整性错误（2处）

### 问题描述
标注时擅自增加了原文没有的字符，违反了**标注铁律**（只能添加标注符号，不得修改原文字符）。

### 错误详情

**L79：幽、厉 → 幽王、厉王**

| 项目 | 内容 |
|-----|------|
| **错误类型** | 插入（标注多字） |
| **原文** | 幽、厉之後 |
| **错误标注** | 〖@幽王〗、〖@厉王〗之後 |
| **正确标注** | 〖@幽〗、〖@厉〗之後 |
| **错误原因** | 擅自增加"王"字（×2处） |

**lint报告摘录**：
```
[1] 插入（标注多字）
    原文  : （空）
    标注  : '王'
    原文上下文 : …则正朔不行於诸侯。幽、厉之後，周室微…
    标注上下文 : …则正朔不行於诸侯。幽王、厉王之後，周室微…

[2] 插入（标注多字）
    原文  : （空）
    标注  : '王'
    原文上下文 : …幽、厉之後，周室微，陪臣执政…
    标注上下文 : …幽王、厉王之後，周室微，陪臣执政…
```

### 修正说明
- **原文表述**：原文使用"幽、厉"是省称，省略了"王"字
- **标注原则**：保持原文字符不变，只添加标注符号
- **正确做法**：标注"幽"和"厉"为人名，不添加"王"字

### 规律总结
参考**025章经验**：025章也发现了"擅自增删原文字符"问题。026章延续了这个问题模式，说明在谥号/帝号标注时容易犯此类错误。

**防范建议**：
1. 谥号省称（如"幽、厉"）直接标注，不补全为"幽王、厉王"
2. 标注前后使用`lint_text_integrity.py`验证原文完整性
3. 遇到省称时，标注省称形式而非完整形式
