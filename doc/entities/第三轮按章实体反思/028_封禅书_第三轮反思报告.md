## 执行摘要

028_封禅书是八书之一，主题为封禅祭祀仪式与宗教制度变迁。本轮反思发现旧格式残留、时长消歧缺失、刑法动词遗漏和生物标注不足是主要问题。

**核心问题**:
1. **旧格式残留**: 12处`⟦◉X⟧`格式需要迁移
2. **时长消歧缺失**: 大量时长标注未加`|时长`消歧（A8规律）
3. **刑法动词遗漏**: 杀/诛/伐/击/破等高频刑法动词遗漏
4. **生物标注不足**: 祭祀用动物（牛/羊/驹/龙/蛇等）遗漏

### 二、时长消歧补充（A8规律，23处）

根据A8规律，所有时长类型必须加`|时长`消歧标注：

**制度周期类**:
- L6: `〖%三年〗不为礼` → `〖%三年|时长〗不为礼` (2处)
- L8: `〖%五载〗一〖^巡狩〗` → `〖%五载|时长〗一〖^巡狩〗`

**经过时长类**:
- L6: `〖%千有馀载〗` → `〖%千有馀载|时长〗`
- L10: `〖%三世〗` → `〖%三世|时长〗` (2处)
- L10: `〖%八世〗` → `〖%八世|时长〗`
- L10: `〖%五世〗` → `〖%五世|时长〗`
- L62: `〖%十二岁〗` → `〖%十二岁|时长〗`
- L128: `六〖%十馀岁〗` → `六〖%十馀岁|时长〗`

**任期时长类**:
- L32: `立三十〖%九年〗而卒` → 需补充
- L160: `朕临〖_天下〗〖%二十有八年〗` → 已有标注

**其他时长**:
- 多处`〖%X年〗`/`〖%X世〗`/`〖%X岁〗`需逐一判断语境

**判断依据**:
- 能否替换为"第X年"且不改变语义 → 是：时长（需`|时长`）
- 是历法/编年体的时间坐标点 → 是：纪年格式（无需消歧）
- 有语法助词"而/矣/毕"修饰 → 通常是时长

**模式1：时长消歧遗漏**
- "三年不为礼" → 制度周期，需`|时长`
- "五载一巡狩" → 制度周期，需`|时长`
- "其後X世" → 经过时长，需`|时长`

**模式2：动词形式刑法词汇遗漏**
- "伐/攻/击/破/灭"等军事动词
- "杀/诛/弑/夷"等处决动词
- A51规律：本纪类章节刑法动词密度高

**模式3：祭祀语境生物遗漏**
